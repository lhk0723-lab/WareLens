# WareLens
AI-powered platform for fashion data analysis and logistics integration.

#1. 프로젝트 소개
WareLens는 사용자의 체형 정보와 보유 의류를 기반으로 맞춤형 의류를 추천하는 AI 기반 웹 서비스이다.
본 PoC의 목적은 서비스 구현에 필요한 핵심 기술의 실현 가능성을 검증하고, MVP(Minimum Viable Product) 수준의 서비스 구축 가능 여부를 확인하는 것이다.

#2. 프로젝트 목표
본 프로젝트에서 검증해야 할 핵심 기술 요소는 다음과 같다.

### 의류 추천

* 의류 이미지로부터 특징 벡터 추출 가능 여부
* 유사한 스타일의 의류 검색 가능 여부
* 이미지 기반 추천 시스템 구축 가능 여부

### 체형 분석

* 전신사진 기반 체형 특징 추출 가능 여부
* 체형 비율 계산 가능 여부
* 체형 기반 사이즈 추천 가능 여부

### 시스템 통합

* Spring Boot와 Python AI 서버 연동 가능 여부
* AI 모델 결과를 웹 서비스에 적용 가능 여부

#3. 기술 스택
##AI 추천 시스템 (CLIP)
*FAISS: 벡터 저장, 벡터 검색 최적화 및 구축

## AI 체형 분석 (MediaPipe)

*Landmark 추출
*BMI 계산

##AI Platform & Integration
*FastAPI
*API Gateway 설계
