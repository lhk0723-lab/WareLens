# WareLens - MediaPipe 기반 AI 체형 분석 및 사이즈 추천 시스템 기획서

## 1. 담당 분야

AI 기반 체형 분석 및 의류 사이즈 추천 시스템 개발

담당 기술
- MediaPipe (Pose Landmarker)
- Python
- FastAPI
- OpenCV

---

## 2. 개발 목표

사용자가 입력한 키, 몸무게 정보와 전신 사진을 활용하여

MediaPipe로 신체 랜드마크를 추출하고 비율을 계산하여

최적의 상의 사이즈 및 체형 맞춤형 핏 스타일을 추천하는 기능 구현

---

## 3. 서비스 시나리오

### Step 1

사용자가 기본 신체 데이터(키, 몸무게) 입력 후 전신 사진 업로드

---

### Step 2

MediaPipe Tasks API(Heavy 모델)를 이용해 33개 포즈 랜드마크 추출

---

### Step 3

추출된 좌표 기반 신체 비율 계산 (어깨 넓이, 상/하체 길이, 골반 비율 등)

---

### Step 4

사이즈 차트 기반 기본 후보군 선정 후 BMI 및 어깨 비율 데이터로 다차원 보정

---

### Step 5

최종 추천 사이즈 및 최적의 핏 유형 판정, 관절 오버레이 이미지 생성 및 반환

---

## 4. 데이터 구조

### 입력 데이터

- 사용자 키 (height_cm)
- 사용자 몸무게 (weight_kg)
- 전신 사진 이미지

---

### 사이즈 차트 스펙 (기준 데이터)

#### S
- Height: 160 ~ 170
- Weight: 50 ~ 60

#### M
- Height: 168 ~ 175
- Weight: 58 ~ 68

#### L
- Height: 173 ~ 182
- Weight: 65 ~ 78

#### XL
- Height: 180 ~ 190
- Weight: 75 ~ 90

---

### 핏 스타일 유형

- SLIM (슬림핏)
- REGULAR (레귤러핏)
- SEMI_OVER (세미오버핏)
- OVER (오버핏)

---

## 5. 추천 및 판정 방식

### 기본 추천

키와 몸무게 기준 인접 사이즈 유연성 점수제 채점

---

### 다차원 보정 로직

#### 1. BMI 보정
- BMI 25 이상 (과체중): 상위 사이즈 조절
- BMI 18.5 미만 (저체중): 하위 사이즈 조절

#### 2. 골격 비율 보정
- 평균 대비 넓은 어깨: 상위 사이즈 조절
- 평균 대비 좁은 어깨: 하위 사이즈 조절

---

### 핏 스타일 판정

- 상/하체 길이 비율 및 골반/어깨 실루엣 계수를 매칭하여 4대 핏 유형 확정

---

## 6. 시스템 구조

사용자

↓ (키/몸무게 입력 & 사진 업로드)

Spring Server

↓ (데이터 프레임 토스)

FastAPI (MediaPipe)

↓ (랜드마크 추출 및 신체 비율 연산)

사이즈/핏 보정 알고리즘

↓ (결과 및 시각화 이미지 생성)

Spring

↓

Frontend (결과 출력)

---

## 7. API 설계

### 체형 분석 요청

POST /api/v1/analyze/body

---

### Request (Multipart Form Data)

{
  "height_cm": 175.0,
  "weight_kg": 70.0,
  "file": "file_binary"
}

---

### Response

{
  "status": "SUCCESS",
  "data": {
    "bmi": 22.86,
    "bmi_grade": "정상",
    "ratios": {
      "shoulder_ratio": 0.225,
      "upper_lower_ratio": 0.684
    },
    "recommendation": {
      "final_size": "M",
      "fit_type": "레귤러핏",
      "reasons": [
        "키/몸무게 기본 후보 M 선정",
        "어깨 비율 표준 범위 반영",
        "종합 결과 레귤러핏 판정"
      ]
    },
    "annotated_image_base64": "base64_encoded_string"
  }
}

## 8. 구현 범위

### 데이터 처리

- 이미지 업로드 및 OpenCV 버퍼 변환
- MediaPipe 연동 랜드마크 추출
- 유클리드 거리 기반 신체 픽셀 거리 계산

### 추천 엔진

- 사이즈 차트 경계값 매칭 스코어링 알고리즘
- BMI 및 어깨 넓이 기반 사이즈 증감 보정 룰 레이어
- 상/하체 실루엣 기반 핏 스타일 분류 파이프라인

### API 및 예외 처리

- FastAPI 엔드포인트 구축
- 랜드마크 신뢰도(Visibility) 미달 시 예외 처리
- Zero Division 등 수학적 에러 방지 방어 코드

### 테스트

- 전신/상반신 이미지 인식 신뢰도 테스트
- BMI 및 골격 수치별 사이즈 보정 정확도 검증
- 비정상 이미지(인물 부재 등) 입력 예외 처리 작동 테스트

## 최종 목표

사용자가 입력한 신체 스펙과 전신 사진을 MediaPipe AI로 분석하여 단순 매칭을 넘어선 개개인의 골격형(어깨, 상하체 비율) 맞춤 상의 사이즈와 최적의 핏을 도출하는 고정밀 분석 시스템을 구축한다. 공모전 MVP에서는 상의(TOP) 스펙트럼을 타깃으로 완벽한 아키텍처를 증명하고, 추후 하의 및 아우터까지 유연하게 확장 가능한 모듈 구조를 지향한다.
