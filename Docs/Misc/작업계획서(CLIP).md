# WareLens - CLIP 기반 취향 분석 의류 추천 시스템 개발 계획

## 담당자

CLIP 기반 의류 추천 시스템 개발

---

# 목표

사용자가 선호하는 의류 이미지 2~3장을 업로드하면 CLIP을 활용하여 사용자의 패션 취향을 분석하고, 유사한 스타일의 상품을 추천하는 기능 구현

단순 이미지 유사도 검색이 아닌, 사용자의 선호 스타일을 반영한 취향 기반 추천 서비스를 목표로 한다.

---

# 서비스 개요

사용자는 평소 좋아하는 의류 이미지 2~3장을 업로드한다.

시스템은 업로드된 이미지들의 특징을 분석하여 사용자의 패션 취향을 벡터 형태로 표현하고, 해당 취향과 유사한 상품을 추천한다.

또한 사용자는 추천 성향을 선택하여 원하는 추천 결과를 받을 수 있다.

---

# 시스템 구조

선호 의류 이미지 2~3장
↓
CLIP 임베딩 생성
↓
취향 벡터 생성 (평균 임베딩)
↓
FAISS 유사도 검색
↓
메타데이터 기반 점수 보정
↓
추천 결과 반환

---

# 개발 단계

## 1. 데이터셋 구축

### 목표

추천 대상이 될 의류 데이터 확보

### 작업 내용

* 의류 이미지 수집
* 카테고리 분류
* 메타데이터 작성
* CSV 생성
* 데이터 검수

---

### 메타데이터 구조

필수

* category
* sub_category
* color
* pattern

선택

* season
* fit

향후 확장

* brand
* price
* material

---

## 2. CLIP 모델 적용

### 목표

의류 이미지를 벡터로 변환

### 작업 내용

* CLIP 모델 적용
* 이미지 임베딩 생성
* 벡터 저장 구조 설계

### 산출물

상품별 임베딩 벡터

예시

hoodie001.jpg

↓

[0.123, -0.543, ...]

---

## 3. 사용자 취향 벡터 생성

### 목표

여러 장의 선호 의류 이미지를 통합하여 사용자 취향 표현

### 작업 내용

사용자가 업로드한 2~3장의 의류 이미지에 대해 각각 임베딩을 생성한다.

생성된 임베딩을 평균하여 하나의 취향 벡터를 생성한다.

예시

이미지 A → 벡터 A

이미지 B → 벡터 B

이미지 C → 벡터 C

↓

취향 벡터

(A + B + C) / 3

---

## 4. 벡터 검색 시스템 구축

### 목표

사용자 취향과 유사한 상품 검색

### 작업 내용

* FAISS 적용
* 벡터 인덱스 구축
* Top-K 검색 구현

### 출력 예시

Top 5 추천

1. black_hoodie_001
2. black_sweatshirt_017
3. gray_hoodie_004
4. black_knit_002
5. navy_sweatshirt_011

---

## 5. 추천 성향(프리셋) 기능

### 목표

사용자가 원하는 추천 기준 선택

### 제공 프리셋(예시)

#### 1. 균형형 (Balanced)

색상 33%

패턴 33%

카테고리 34%

---

#### 2. 색상 중시형 (Color Focus)

색상 60%

패턴 20%

카테고리 20%

---

#### 3. 스타일 중시형 (Style Focus)

색상 20%

패턴 40%

카테고리 40%

---

### 추천 점수 계산

최종 점수 =

CLIP 유사도 +
색상 점수 +
패턴 점수 +
카테고리 점수

각 프리셋에 따라 반영 비율을 다르게 적용한다.

---

## 6. 추천 품질 개선

### 목표

추천 결과 정확도 향상

### 작업 내용

* 색상 기반 보정
* 패턴 기반 보정

추천 결과

* 블랙 맨투맨
* 다크 그레이 맨투맨
* 블랙 니트
