# WareLens 프로젝트 진행 현황 및 향후 계획

## 프로젝트 개요

사용자가 선호하는 의류 이미지를 업로드하면 취향에 맞는 의류를 추천받고,

전신 사진을 업로드하면 체형 분석을 통해 상의 사이즈를 추천받을 수 있는 패션 추천 서비스 개발

## 프로젝트 운영 원칙

1. 데이터 수집 및 라벨링은 전원 참여

2. 구현 과정에서 발생하는 주요 변경사항은 팀원 전체 공유

3. UI는 프론트엔드 담당자가 초안을 작성하고 팀 전체가 검토 후 확정

4. 기능 구현보다 서비스 완성도와 정확도 향상을 우선

5. 6월 말 구현 완료, 7월 중순 최종 제출 버전 완성을 목표로 진행
