# WareLens - Frontend(React) 서비스 구축 기획서


## 1. 담당 분야

사용자 서비스 화면 및 웹 인터페이스 개발


담당 기술

- React
- TypeScript
- Tailwind CSS
- REST API 연동


---

# 2. 개발 목표

사용자가 업로드한 이미지와 입력 정보를 기반으로

AI 분석 결과를 확인하고

개인 맞춤 의류 추천 서비스를 이용할 수 있는

사용자 중심 웹 인터페이스 구현


Frontend에서 사용자 입력부터 AI 추천 결과 확인까지

전체 서비스 흐름을 연결하는 화면 개발을 목표로 한다.


---

# 3. 서비스 시나리오


## Step 1

사용자가 서비스 접속


서비스 소개 및 추천 시작 화면 제공


---

## Step 2

사용자가 선호하는 의류 이미지 업로드


기능

- 이미지 업로드
- 이미지 Preview
- 이미지 삭제


전달 데이터

- 사용자 취향 이미지


---

## Step 3

사용자 신체 정보 입력


입력 정보

- 키
- 몸무게
- 전신 사진


기능

- 사용자 입력값 검증
- 분석 요청 데이터 준비


---

## Step 4

사용자가 추천 성향 선택


추천 옵션

- 균형형
- 색상 중심
- 스타일 중심


---

## Step 5

AI 분석 결과 출력


제공 정보

- 취향 분석 결과
- 추천 스타일
- 추천 상품
- 체형 분석 결과
- 추천 사이즈
- 추천 Fit


---

# 4. 화면 구성


## Home Page

서비스 메인 화면


기능

- 서비스 소개
- AI 의류 추천 기능 안내
- 추천 시작 버튼 제공


---

## Profile Page

사용자 정보 입력 및 이미지 업로드 화면


기능

- 선호 의류 이미지 업로드
- 전신 사진 업로드
- 사용자 신체 정보 입력
- 이미지 Preview


입력 데이터

- 키
- 몸무게
- 사용자 이미지


---

## Analysis Page

AI 분석 결과 확인 화면


제공 정보


취향 분석

- 사용자 스타일 정보
- 선호 색상
- 추천 스타일


체형 분석

- 체형 결과
- 추천 사이즈
- 추천 Fit


---

## Recommend Page

최종 추천 결과 화면


제공 정보

- 추천 의류 목록
- 상품 이미지
- 추천 점수
- 추천 사이즈
- 추천 스타일
