# WareLens 역할 분담안 비교

==================================================
안 A. AI 분리형
==================================================

1. AI 추천 시스템 (CLIP)
2. AI 체형 분석 (MediaPipe)
3. Backend (Spring)
4. Frontend + Integration

--------------------------------------------------
장점
--------------------------------------------------

1. 업무량 분산

- AI 업무를 두 명이 나눠 가짐
- 개인 부담 감소
- 병렬 개발 가능

2. AI 경험 인원 증가

- CLIP 담당 1명
- MediaPipe 담당 1명

AI 관련 경험자가 2명 생김

3. 개발 속도 향상 가능

- 추천 시스템 개발
- 체형 분석 개발

동시에 진행 가능

4. 역할이 명확함

- 각자 담당 영역이 뚜렷함
- 책임 소재가 명확함

--------------------------------------------------
단점
--------------------------------------------------

1. AI 구조 통합 필요

- CLIP 결과
- MediaPipe 결과

최종 추천 결과로 합치는 과정 필요

2. 설계 방향이 달라질 수 있음

예시

CLIP 담당:
"이렇게 하자"

MediaPipe 담당:
"저렇게 하자"

→ 조율 필요

3. AI 담당자 역량 차이 발생 가능

한쪽은 빠르게 진행
한쪽은 느리게 진행

→ 일정 차이 발생

4. Frontend + Integration 업무 증가

React
API 연동
Docker
배포
통합 테스트

모두 담당

--------------------------------------------------
추천 상황
--------------------------------------------------

- AI 해보고 싶은 사람이 2명 이상
- CLIP / MediaPipe를 각각 경험해보고 싶은 경우
- AI 업무를 분산하고 싶은 경우

==================================================
안 B. AI 통합형
==================================================

1. AI 총괄 (CLIP + MediaPipe)
2. Backend (Spring)
3. Frontend (React)
4. Integration / AI Platform

--------------------------------------------------
장점
--------------------------------------------------

1. 설계 일관성 확보

AI 구조를 한 명이 관리

→ 방향성이 흔들리지 않음

2. 통합이 쉬움

추천 결과
+
체형 분석 결과

한 사람이 설계

→ API 설계가 단순

3. 현재 PoC와 자연스럽게 연결

현재 진행한

- CLIP 검증
- MediaPipe 검증
- FastAPI 검증

흐름을 그대로 이어갈 수 있음

4. 역할 균형이 좋음

AI
Backend
Frontend
Integration

업무 영역이 깔끔하게 나뉨

5. 배포/통합 담당 존재

Docker
API 테스트
자동화
배포

전담 담당자 존재

--------------------------------------------------
단점
--------------------------------------------------

1. AI 담당 업무량 증가

CLIP
MediaPipe
FastAPI

모두 담당

2. AI 담당자가 병목이 될 수 있음

AI 작업이 늦어지면

Backend
Frontend

연동 일정도 밀릴 수 있음

3. AI 경험 인원이 1명

다른 팀원은 AI 개발 경험이 적게 남음

--------------------------------------------------
추천 상황
--------------------------------------------------

- AI 희망자가 1명 이하
- 현재 PoC를 진행한 사람이 AI를 계속 맡는 경우
- 공모전 MVP를 빠르게 완성하는 것이 목표인 경우

==================================================
결론
==================================================

AI 희망자 2명 이상

→ 안 A 추천

→ 안 B가 조금 더 현실적
